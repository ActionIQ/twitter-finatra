object KafkaConsumerTracer

  18.  def trace[K, V, T](records: ConsumerRecords[K, V], configMap: Map[String, AnyRef] = Map.empty)(f: => T): T

    Setup tracing from the Kafka records that were consumed and perform future operations in the provided continuation function f.

    Setup tracing from the Kafka records that were consumed and perform future operations in the provided continuation function f. We will only start a trace if one of the TraceIds in the ConsumerRecords was sampled or flagged as debug.

    K

    type of Key.

    V

    type of Value.

    T

    return type of the continuation function f.

    records

    consumed from the Kafka consumer.

    configMap

    configuration used for the Kafka consumer. Defaults to Map.empty. This is required to record the following annotations,

    • KafkaConsumerConfig.FinagleDestKey
    • org.apache.kafka.clients.consumer.ConsumerConfig.CLIENT_ID_CONFIG
    • org.apache.kafka.clients.consumer.ConsumerConfig.GROUP_ID_CONFIG
    f

    continuation function to perform operations on the consumed records. The function will have tracing setup properly so any RPC calls made will be traced appropriately.

    returns

    the return value from the continuation function f.

    Example:

    1. val records = consumer.poll(5.seconds.asJava)
FinagleKafkaConsumerTracer.trace(records) {
  // make more RPCs
}
